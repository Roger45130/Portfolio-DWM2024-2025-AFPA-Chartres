Roger-Pierre LE GRASSE

Développeur Web et Web Mobile Junior

En 2023, je me suis lancé le défi ambitieux de devenir développeur web et web mobile. Cette reconversion professionnelle a été motivée par un problème de santé rendant mon ancien emploi incompatible avec mon état de santé.

En septembre 2024, j'ai intégré une formation à l'A.F.P.A de Chartres (28) dans le but d'obtenir un titre de Développeur Web et Web Mobile en juin 2025.


Avant de me lancer dans le Développement Web et Web Mobile

De transport en commun au Développement Web et Web Mobile.

Avant de m'orienté dans le Développement Web et Web Mobile, j'étais dans le domaine du transport de voyageur pour une société qui se trouve dans le Loir-et-Cher (41) qui se situe dans la ville d'Ouzouer-le-Marché (Beauce-la-Romaine).

Pour être conducteur de transport routier de voyageur (c'est le nom officiel dans la profession) j'ai passé un titre professionnelle au centre de l'A.F.P.A. de la ville de Châteauroux dans le département de l'indre (36).

J'ai d'ailleurs exercer la profession de conducteur de transport de voyageurs pendant une durée de 5 ans.
